# AGENTS.md

Instructions for AI coding agents working on this Slidev presentation template.

## 📋 Required Reading

**Before making any changes, read these files in order:**

1. **[.github/instructions/slides.instructions.md](.github/instructions/slides.instructions.md)** - Complete guidelines for creating and editing Slidev presentations
2. **[README.md](README.md)** - Project overview, structure, and commands
3. **[package.json](package.json)** - Available npm scripts and dependencies

## ⚡ Quick Reference

```bash
npm install                    # Setup
npm run dev:example           # Run example presentation
npm run format                # Format code (ALWAYS run after changes)
```

## 🎯 Critical Rules

1. **Always read `.github/instructions/slides.instructions.md` before creating or modifying presentations**
2. **Always run `npm run format` as the final step after any changes**
3. Folder names must match markdown filenames (e.g., `slides/topic-name/topic-name.md`)
4. Update `package.json` scripts AND `README.md` when adding presentations
5. Use Context7 MCP tools for accurate library documentation

## 📁 Key Files

- `.github/instructions/slides.instructions.md` - Slidev conventions and guidelines
- `README.md` - Human-readable documentation
- `package.json` - All npm scripts and dependencies
- `biome.json` - Code formatting configuration
